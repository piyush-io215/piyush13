Web Technology Module

A 7-Day Comprehensive Journey into Modern Web Development

Day 01

Basic Text Elements

Mastering the fundamental building blocks of HTML structure and text formatting.

  • <p> Paragraph structures
  • <b> & <i> Text emphasis
  • <h1-h6> Semantic headings
  • <marquee> Legacy animations
Day 02

Media & Data Tables

Integrating visual assets and organizing complex information into tables.

  • <img> Responsive image handling
  • <a> Navigation and Hyperlinks
  • <video> & <audio> Multimedia
  • <table> Data structure and borders
Day 03

User Input & Forms

Designing interactive interfaces to capture and validate user information.

  • <form> Action and Method attributes
  • Input types (text, email, password, checkbox)
  • <textarea> for feedback loops
  • Submit buttons and accessible labels
Day 04

Semantic HTML

Optimizing page structure for SEO and screen readers using modern standards.

  • <header> & <nav> Site navigation
  • <main> & <section> Content flow
  • <footer> Meta information
  • Accessibility (ARIA) basics
Day 05

CSS Box Model & Flexbox

Controling the layout and spacing of every element on the screen.

  • Box Model: Padding vs Margin vs Border
  • Flexbox: Direction, Wrap, and Alignment
  • CSS Variables for consistent branding
  • Responsive container design
Day 06

JavaScript Fundamentals

Injecting logic and interactivity into static pages using scripting.

  • Arithmetic and Logical operators
  • Variable scopes (let, const, var)
  • Functions and Arrow functions
  • Conditional logic (if/else, switch)
Day 07

Final Project: To-Do List

The capstone project combining all previous skills into a working application.

  • DOM Selection and Manipulation
  • Event Listeners (Click, Keydown)
  • Dynamic Element creation
  • State management (Add/Delete tasks)
