Components, Auto Layout & Project Kickoff

Learning reusable design systems through components and kicking off a real-world project.

  • Started Project: Red Bull Sugar Free landing page

💡 Component = Parent  |  Instance = Child  —  Instances inherit but can be overridden individually.

Day 05

Glass Effect, UI vs UX & Case Study

Explored the glassmorphism effect and dived into the psychology behind UX design decisions.

  • Learned the Glass Effect (Glassmorphism) in Figma
  • Assignment: apply Glass Effect in project — usable in task bars & buttons
  • Deliverable: Design two versions — Main UI + Glass Effect UI
  • UI deals with Design  |  UX deals with Psychology
  • Case Study — Why does a phone dial start 1→9 but a calculator starts 9→1?
  • Discussed Project Evaluation criteria
🧠 UX Insight: Phone keypads start from 1 (top) because of early telephone research on user habits. Calculators inherited the adding-machine layout where larger numbers were used more frequently and placed at the top for speed.
